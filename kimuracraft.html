Welcome to KimuraCraft

KimuraCraft is our free to play, flagship Minecraft SMP (Survival Multiplayer) server, designed to provide an unparalleled gaming experience. Dive into a world of creativity, adventure, and community. Whether you're a builder, explorer, or warrior, KimuraCraft has something for everyone. Join us and be part of an epic journey!

Getting Started

To join KimuraCraft, as it is a modded Minecraft server, you'll need a few things. Don't worry, our team is here to help if you have any trouble setting up!

  • A valid Minecraft Java Edition account
  • Install the latest version of CurseForge; here's the link: CurseForge
  • Download MC Eternal 2 and add it to the "Mods" section of your Minecraft folder. You can access this folder in game.
  • Install this VoiceChat mod into your mod folder (Optional): VoiceChat Mod
  • Connect to the server using the IP: kimuracraft.minecraft.best
